Project Description
The Hospital Management System (HMS) is a comprehensive web-based solution designed to modernize and streamline healthcare administration. Developed as a 6-credit hour Final Year Project (CSI-630), this system automates manual processes to improve operational efficiency in medical environments.
This project integrates core competencies from my BS Computer Science curriculum. Specifically, it applies principles from Web Design and Development (CSI-506) and Database Systems (CSI-408) to create a functional, data-driven application.
Key Features
Patient Management: 
Facilitates digital registration and maintains organized medical records.
Doctor Scheduling:
Manages availability and coordinates patient appointments.
Administrative Dashboard: Provides an overview of hospital statistics and centralized data management.
Secure Authentication: Implements role-based access control for administrators, doctors, and staff to ensure data security.
Technical Stack
Frontend: HTML5, CSS3, JavaScript.
Backend: PHP.
Database: MySQL.
